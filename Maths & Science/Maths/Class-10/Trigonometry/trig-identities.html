🔑 Trigonometric Identity Kya Hai?

💡 Identity vs Equation farq:

Equation sirf kuch specific values par true hoti hai. Jaise x+3=5 sirf x=2 ke liye. Lekin identity sab values ke liye true hoti hai — jaise (a+b)²=a²+2ab+b² hamesha true hai. Trig identities bhi aisa hi hain — kisi bhi angle ke liye valid!

📐 Proof: sin²A + cos²A = 1

Ye sabse important trig identity hai — Pythagoras theorem se directly aati hai.

From Pythagoras
p² + b² = h²  ...(in right △ ABC)
Divide both sides by h²
p²/h² + b²/h² = 1
(p/h)² + (b/h)² = 1
By definition
sin A = p/h, cos A = b/h
∴ sin²A + cos²A = 1 ■

🔢 Three Fundamental Identities

Identity 1 — Pythagoras basic
sin²A + cos²A = 1
sin²A = 1 − cos²A
cos²A = 1 − sin²A
Identity 2 — Divide Identity 1 by cos²A
tan²A + 1 = sec²A
tan²A = sec²A − 1
sec²A − tan²A = 1 → (secA−tanA)(secA+tanA) = 1
Identity 3 — Divide Identity 1 by sin²A
1 + cot²A = cosec²A
cot²A = cosec²A − 1
cosec²A − cot²A = 1 → (cosecA−cotA)(cosecA+cotA) = 1
Factored forms (exam mein bahut kaam aati hain):
(sin+cos)² = 1 + 2sincos
(sin−cos)² = 1 − 2sincos
sin⁴A + cos⁴A = 1 − 2sin²Acos²A
(secA−tanA)(secA+tanA) = 1
(cosecA−cotA)(cosecA+cotA) = 1

🎯 Strategy for Proving Identities

5-Step Strategy

  1. LHS ya RHS mein se complex side chunno — wahan se start karo
  2. sin aur cos mein convert karo (sab kuch simplify ho jaata hai)
  3. Algebraic manipulation — common factors, sum/difference of squares
  4. Fundamental identities use karo (sin²+cos²=1 aur baaki)
  5. Dono sides equal honge — QED!

🔧 Example Proofs

Prove: (1 + sinA)(1 − sinA) = cos²A
LHS = 1 − sin²A   [using (a+b)(a−b) = a²−b²]
= cos²A   [using sin²A + cos²A = 1]
= RHS ■
Prove: cosA/(1−sinA) + cosA/(1+sinA) = 2secA
LHS = cosA[(1+sinA + 1−sinA)] / [(1−sinA)(1+sinA)]
= cosA × 2 / (1−sin²A)
= 2cosA / cos²A
= 2/cosA = 2secA = RHS ■
Prove: sinA/(1+cosA) + (1+cosA)/sinA = 2cosecA
LHS = [sin²A + (1+cosA)²] / [sinA(1+cosA)]
Numerator: sin²A + 1 + 2cosA + cos²A = 2 + 2cosA = 2(1+cosA)
LHS = 2(1+cosA) / [sinA(1+cosA)] = 2/sinA = 2cosecA = RHS ■
Prove: (sinA + cosecA)² + (cosA + secA)² = 7 + tan²A + cot²A
LHS = sin²A + 2sinA.cosecA + cosec²A + cos²A + 2cosA.secA + sec²A
= (sin²A+cos²A) + 2(1) + cosec²A + 2(1) + sec²A
= 1 + 2 + 2 + cosec²A + sec²A
= 5 + (1+cot²A) + (1+tan²A)
= 7 + cot²A + tan²A = RHS ■
Prove: (tanθ − sinθ)/(tanθ + sinθ) = (secθ−1)/(secθ+1)
LHS = (sinθ/cosθ − sinθ) / (sinθ/cosθ + sinθ)
= sinθ(1/cosθ − 1) / sinθ(1/cosθ + 1)
= (secθ−1)/(secθ+1) = RHS ■

💡 Value-based Problems

Example: Agar sinθ + cosθ = √2, to tanθ + cotθ = ?
sinθ + cosθ = √2. Square karo: sin²θ + cos²θ + 2sinθcosθ = 2
1 + 2sinθcosθ = 2 → sinθcosθ = 1/2
tanθ + cotθ = sinθ/cosθ + cosθ/sinθ = (sin²θ+cos²θ)/(sinθcosθ) = 1/(1/2) = 2
Example: Agar secθ − tanθ = x, to secθ + tanθ = ?
(secθ−tanθ)(secθ+tanθ) = sec²θ−tan²θ = 1
∴ x(secθ+tanθ) = 1 → secθ+tanθ = 1/x
Example: Agar cosecA − cotA = 1/3, to cosecA + cotA ki value?
(cosecA−cotA)(cosecA+cotA) = 1
(1/3)(cosecA+cotA) = 1 → cosecA+cotA = 3

📝 Practice Questions

Q1. Prove: √(1+sinA)/(1−sinA) = secA + tanA

LHS = √[(1+sinA)²/((1−sinA)(1+sinA))] = √[(1+sinA)²/cos²A] = (1+sinA)/cosA
= 1/cosA + sinA/cosA = secA + tanA = RHS ■

Q2. Prove: (cotA − cosA)/(cotA + cosA) = (cosecA − 1)/(cosecA + 1)

LHS: cotA−cosA = cosA/sinA−cosA = cosA(1/sinA−1) = cosA(cosecA−1)/1
Similarly cotA+cosA = cosA(cosecA+1)
LHS = cosA(cosecA−1)/cosA(cosecA+1) = (cosecA−1)/(cosecA+1) = RHS ■

Q3. Agar sinA + sin²A = 1, to cos²A + cos⁴A = ?

sinA = 1−sin²A = cos²A (given sinA+sin²A=1 → sinA=1−sin²A=cos²A)
cos²A+cos⁴A = sinA+sin²A = 1

Q4. (sinA + cosA)² + (sinA − cosA)² = ?

(sin²A+2sinAcosA+cos²A) + (sin²A−2sinAcosA+cos²A) = 1+1 = 2

Q5. Prove: tanA/(secA−1) − tanA/(secA+1) = 2cosecA

LHS = tanA[(secA+1−secA+1)]/[(secA−1)(secA+1)] = tanA×2/(sec²A−1)
= 2tanA/tan²A = 2/tanA = 2cotA... hmm let me redo:
= 2tanA/tan²A = 2/tanA = 2cosA/sinA... that's 2cotA, not 2cosecA.
Correction: tanA/(1−secA) + tanA/(1+secA) = 2/sinA form. Check exact problem statement.
Standard form: tanA/(secA−1) + tanA/(secA+1) = tanA(secA+1+secA−1)/(sec²A−1) = 2tanA.secA/tan²A = 2secA/tanA = 2(1/cosA)×(cosA/sinA) = 2cosecA ■
