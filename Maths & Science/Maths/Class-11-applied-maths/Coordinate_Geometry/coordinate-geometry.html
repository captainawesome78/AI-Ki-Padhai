📐 Coordinate Geometry

Slope, line equations, distance formula aur section formula — sab ek jagah!

📏 1. Slope aur Inclination

Ek line ka slope batata hai ki woh kitni tezi se upar/neeche jaati hai. Ramp, road, graph — sab jagah slope use hota hai.

Slope Formula: m = (y₂ − y₁) / (x₂ − x₁) = Rise / Run = tan θ Jahan θ = angle line aur x-axis ke beech (inclination angle) m > 0 → line upar jaati hai (left se right) m < 0 → line neeche jaati hai m = 0 → horizontal line m undefined → vertical line (x = constant)
Example: Slope Nikaalte Hain

Do points A(2, 3) aur B(6, 11) ke beech slope nikalo. Inclination angle bhi batao.

1
m = (y₂−y₁)/(x₂−x₁) = (11−3)/(6−2) = 8/4 = 2
2
m = tan θ = 2 → θ = tan⁻¹(2) ≈ 63.43°

Parallel aur Perpendicular Lines

Parallel lines: m₁ = m₂ (same slope, different y-intercept) Perpendicular lines: m₁ × m₂ = −1 (slopes ka product = −1) ya: m₂ = −1/m₁ (negative reciprocal) Example: Line 1 ka slope = 3/4 → Parallel line ka slope = 3/4 → Perpendicular line ka slope = −4/3
💡 Angle between two lines: tan α = |(m₁ − m₂)/(1 + m₁m₂)| — agar answer = 0, lines parallel; agar denominator = 0, lines perpendicular!

📝 2. Straight Line ke Equations

Ek line ko describe karne ke 4 main forms hain. Sab equivalent hain — situation ke hisaab se sahi form use karo.

1. Slope-Intercept Form

y = mx + c

m = slope, c = y-intercept (jahan line y-axis ko cut karti hai). Sabse zyada use hone wali form!

2. Point-Slope Form

y − y₁ = m(x − x₁)

Jab ek point aur slope diya ho. (x₁,y₁) us point ke coordinates hain.

3. Two-Point Form

(y−y₁)/(y₂−y₁) = (x−x₁)/(x₂−x₁)

Jab do points diye hoon. Slope nikaalte hain phir point-slope use karte hain.

4. Intercept Form

x/a + y/b = 1

a = x-intercept, b = y-intercept. Jab dono intercepts diye hon.

General Form

General Form: ax + by + c = 0 Slope = −a/b x-intercept = −c/a (y=0 rakho) y-intercept = −c/b (x=0 rakho) Kisi bhi form ko general form mein convert kar sakte hain!
Example 1: Slope-Intercept se General Form

Line y = 3x − 5 ko general form mein likho. Dono intercepts nikalo.

1
General form: 3x − y − 5 = 0 (a=3, b=−1, c=−5)
2
x-intercept: y=0 → 3x=5 → x = 5/3
3
y-intercept: x=0 → −y=5 → y = −5 (yaani c = −5)
Example 2: Two Points se Line Equation

Points P(1, 2) aur Q(4, 8) se guzarne wali line ki equation nikalo.

1
m = (8−2)/(4−1) = 6/3 = 2
2
Point-slope: y − 2 = 2(x − 1)
3
y − 2 = 2x − 2 → y = 2x (line origin se guzarti hai!)

Do Lines ka Intersection

Do lines ka intersection: simultaneous equations solve karo L1: a₁x + b₁y + c₁ = 0 L2: a₂x + b₂y + c₂ = 0 Cramer's rule ya substitution se solve karo. Example: x + y = 5 aur 2x − y = 1 → Add: 3x = 6 → x = 2, y = 3 → Intersection point: (2, 3)

📍 3. Distance aur Section Formula

Distance Formula

Do points P(x₁,y₁) aur Q(x₂,y₂) ke beech: d = √[(x₂−x₁)² + (y₂−y₁)²] (Pythagoras theorem ka use! Horizontal aur vertical distances ka hypotenuse)
Example: Distance Nikaalte Hain

A(3, −4) aur B(−1, 2) ke beech distance nikalo.

1
d = √[(-1-3)² + (2-(-4))²]
2
= √[(-4)² + (6)²] = √[16 + 36] = √52
3
= 2√13 ≈ 7.21 units

Section Formula

Point P(x,y) jo A(x₁,y₁) aur B(x₂,y₂) ko m:n mein divide karta hai: Internal Division: x = (mx₂ + nx₁)/(m+n) y = (my₂ + ny₁)/(m+n) External Division: x = (mx₂ − nx₁)/(m−n) y = (my₂ − ny₁)/(m−n) Midpoint (m=n=1): x = (x₁+x₂)/2, y = (y₁+y₂)/2
Example: Section Formula

A(2,5) aur B(8,−1) ko join karne wali line ko point P, 2:1 mein internally divide karta hai. P nikalo.

1
m=2, n=1, (x₁,y₁)=(2,5), (x₂,y₂)=(8,−1)
2
x = (2×8 + 1×2)/(2+1) = (16+2)/3 = 18/3 = 6
3
y = (2×(−1) + 1×5)/(2+1) = (−2+5)/3 = 3/3 = 1
4
P = (6, 1)

Point se Line ki Distance

Point P(x₀,y₀) se line ax + by + c = 0 ki perpendicular distance: d = |ax₀ + by₀ + c| / √(a² + b²) Example: Point (3,4) se line 3x − 4y + 5 = 0 ki distance: d = |3×3 − 4×4 + 5| / √(9+16) = |9−16+5| / 5 = |−2| / 5 = 2/5 = 0.4 units
