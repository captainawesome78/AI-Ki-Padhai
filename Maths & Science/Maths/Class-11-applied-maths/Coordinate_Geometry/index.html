Topic 1

📏 Slope aur Inclination

Line ka slope uski steepness batata hai. Inclination angle se slope nikalta hai!

  • Slope m = tan θ = (y₂−y₁)/(x₂−x₁)
  • Parallel lines: m₁=m₂
  • Perpendicular: m₁×m₂=−1

Topic 2

📝 Line ke Equations

Ek line ko multiple tarike se represent kar sakte hain — slope-intercept, point-slope, intercept form!

  • y = mx + c (slope-intercept)
  • y−y₁ = m(x−x₁) (point-slope)
  • x/a + y/b = 1 (intercept form)
  • ax + by + c = 0 (general form)

📍 Distance aur Section Formula

Do points ke beech distance nikalna aur line segment ko kisi ratio mein divide karna!

  • Distance = √[(x₂−x₁)²+(y₂−y₁)²]
