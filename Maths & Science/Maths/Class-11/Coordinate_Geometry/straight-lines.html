Straight Lines 📏

Ek point se ek line draw karo — infinite lines possible hain. Do points do toh exactly ek line banti hai. Yahi hai geometry ka beauty!

🗺️ GPS aur Lines

Jab GPS tumhe kisi jagah ka rasta batata hai — woh straight roads actually coordinate geometry ki lines hain! Ek city map par har road ek line equation se represent hoti hai. Ek road ki steepness (slope) batata hai woh kitni chadhai hai ya dhalaaan hai. Signals pe perpendicular roads (90° angle wali) — unke slopes ka product −1 hota hai. Yahi saari cheezein hum is chapter mein seekhte hain!

1. Slope (Dhalaaan) — m kya hai?

Slope batata hai line kitni "tilted" hai — kitni steep ya flat hai. Do points (x₁,y₁) aur (x₂,y₂) given ho toh slope:

m = (y₂ − y₁) / (x₂ − x₁) = Rise / Run

Angle θ se slope: m = tan θ (jahan θ x-axis ke saath angle hai)

  • m > 0 → line upar jaati hai (left to right) — positive slope
  • m < 0 → line neeche jaati hai — negative slope
  • m = 0 → horizontal line — koi slope nahi
  • m = undefined → vertical line (x₁=x₂)

Alag alag slopes ka visual

✅ A(2,3) aur B(5,9) ke beech slope nikalo

m = (9−3)/(5−2) = 6/3 = 2

matlab: x mein 1 unit badhne par y mein 2 unit badhti hai — yeh moderate slope hai.

2. Parallel aur Perpendicular Lines

Parallel lines: Dono lines ka slope equal hota hai. Woh kabhi nahi milti.

m₁ = m₂ (parallel lines)

Perpendicular lines: Dono slopes ka product = −1. Yeh 90° par milti hain.

m₁ × m₂ = −1 → m₂ = −1/m₁ (perpendicular)

Trick: Perpendicular slope = original ka negative reciprocal. Agar m=2 hai toh perpendicular slope = −1/2.

3. Line ke Different Forms

Ek straight line ko alag alag situations mein alag alag equations se likha ja sakta hai. Yeh saari forms same line ko represent karti hain — bas starting information alag hoti hai:

① Slope-Intercept Form

Slope aur y-intercept given. Sabse common aur useful form.

y = mx + c

m = slope, c = y-intercept (jahan line y-axis ko kaati hai)

② Point-Slope Form

Ek point (x₁,y₁) aur slope m given ho toh use karo.

y − y₁ = m(x − x₁)

Real life: direction aur ek location know ho toh path nikalna.

③ Two-Point Form

Do points given ho aur slope calculate karke point-slope use karo.

(y−y₁)/(y₂−y₁) = (x−x₁)/(x₂−x₁)

Do locations pata ho toh line ka equation nikalna.

④ Intercept Form

x-intercept (a) aur y-intercept (b) dono given ho.

x/a + y/b = 1

Jab line dono axes par kahan milti hai yeh pata ho.

⑤ Normal Form

Origin se perpendicular distance p aur angle ω given.

x·cosω + y·sinω = p

Advanced form — physics mein projection problems mein aata hai.

⑥ General Form

Ax + By + C = 0. Kisi bhi line ko is form mein likha ja sakta hai.

Ax + By + C = 0

Slope = −A/B, y-intercept = −C/B

✅ Point-Slope: (3,−2) se guzarne wali line jiska slope 4 hai

y − (−2) = 4(x − 3)
y + 2 = 4x − 12
y = 4x − 14 ya 4x − y − 14 = 0

✅ Two Points: A(1,2) aur B(4,8) se line ka equation

m = (8−2)/(4−1) = 6/3 = 2
y − 2 = 2(x − 1) → y = 2x (y-intercept = 0, line passes through origin!)

4. Distance Formulas

Point se line ki perpendicular distance: Point P(x₀,y₀) aur line Ax+By+C=0 ke beech:

d = |Ax₀ + By₀ + C| / √(A² + B²)

Do parallel lines ke beech distance: Ax+By+C₁=0 aur Ax+By+C₂=0 ke beech:

d = |C₁ − C₂| / √(A² + B²)

✅ Point (3,4) ki line 3x − 4y + 5 = 0 se distance nikalo

A=3, B=−4, C=5, x₀=3, y₀=4
d = |3(3) + (−4)(4) + 5| / √(9+16)
= |9 − 16 + 5| / √25 = |−2| / 5 = 2/5

5. Two Lines ka Angle

Jab do lines milti hain unke beech ka angle θ nikalna ho:

tan θ = |(m₁ − m₂) / (1 + m₁m₂)|

Agar m₁m₂ = −1 → lines perpendicular (θ = 90°). Agar m₁ = m₂ → lines parallel (θ = 0°).

6. Practice Questions

Q1. A(−1,3) aur B(4,−2) se guzarne wali line ka slope aur equation nikalo.

Solution

m = (−2−3)/(4+1) = −1. y−3=−1(x+1) → y=−x+2 → x+y−2=0

Q2. 3x + 4y = 12 ki slope aur intercepts nikalo.

Solution

x/4 + y/3 = 1 → x-intercept=4, y-intercept=3. Slope = −3/4

Q3. y = 3x + 5 ke parallel aur perpendicular line jo (2,1) se guzre — equations nikalo.

Solution

Parallel (m=3): y−1=3(x−2) → y=3x−5. Perpendicular (m=−1/3): y−1=−1/3(x−2) → x+3y=5

Q4. Point (5,6) aur line 2x − y + 4 = 0 ke beech perpendicular distance nikalo.

Solution

d = |2(5)−6+4|/√(4+1) = |8|/√5 = 8/√5 = 8√5/5

Q5. Lines y=2x+3 aur y=−x+1 ke beech angle nikalo.

Solution

tan θ = |(2−(−1))/(1+2×(−1))| = |3/(1−2)| = |3/−1| = 3 → θ = arctan(3) ≈ 71.6°
