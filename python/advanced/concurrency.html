Concurrency ka matlab hai ek hi samay par multiple tasks ko manage karna ‚Äî jaise ek program ek saath multiple users ko serve kare. Python me concurrency achieve karne ke main tareeke hain:

  • Threads (Lightweight parallel tasks)
  • Multiprocessing (Real parallelism using multiple CPU cores)
  • Asyncio (Single-threaded but non-blocking async tasks)

2Ô∏è‚É£ Threading ‚Äî Ek hi process ke andar multiple lightweight threads

Threading use karte hain jab aapko I/O bound tasks perform karne hote hain ‚Äî jaise file read/write, network calls, etc. 


import threading
import time

def task(name):
    print(f"Task {name} start ho gaya ‚è≥")
    time.sleep(2)
    print(f"Task {name} complete ‚úÖ")

t1 = threading.Thread(target=task, args=("A",))
t2 = threading.Thread(target=task, args=("B",))

t1.start()
t2.start()

t1.join()
t2.join()

print("Sab threads complete üéâ")

3Ô∏è‚É£ Multiprocessing ‚Äî True Parallelism üí™

Threading me Python‚Äôs GIL (Global Interpreter Lock) ki wajah se ek samay me sirf ek thread execute hota hai. Multiprocessing is limitation ko todta hai ‚Äî multiple processes use karta hai jo alag CPU cores par run hote hain. 


from multiprocessing import Process
import os
import time

def work():
    print(f"Process {os.getpid()} start hua")
    time.sleep(2)
    print(f"Process {os.getpid()} khatam hua")

if __name__ == "__main__":
    p1 = Process(target=work)
    p2 = Process(target=work)
    p1.start()
    p2.start()
    p1.join()
    p2.join()
    print("Done ‚úÖ")

4Ô∏è‚É£ Asyncio ‚Äî Modern Asynchronous Python ‚ö°

Asyncio ek single-threaded, single-process event loop hota hai jisme aap asynchronous functions likh sakte ho jo non-blocking hote hain. 


import asyncio

async def fetch_data(id):
    print(f"Fetching data {id}...")
    await asyncio.sleep(2)
    print(f"Data {id} ready ‚úÖ")

async def main():
    await asyncio.gather(fetch_data(1), fetch_data(2), fetch_data(3))

asyncio.run(main())

üëâ Output: sab 3 tasks parallelly run karenge using async await.

6Ô∏è‚É£ Real-life Example: Concurrent Web Scraper

Ye ek simple asyncio-based web scraper example hai jo ek saath multiple URLs fetch karta hai. 


import asyncio
import aiohttp

urls = [
    "https://example.com",
    "https://python.org",
    "https://aikipadhai.com"
]

async def fetch(session, url):
    async with session.get(url) as response:
        print(f"{url} => {response.status}")

async def main():
    async with aiohttp.ClientSession() as session:
        tasks = [fetch(session, url) for url in urls]
        await asyncio.gather(*tasks)

asyncio.run(main())

üí° Summary

  • Threading: Lightweight, great for I/O-bound work.
  • Multiprocessing: Heavy-duty, true CPU parallelism.
  • Asyncio: Modern async code with simple syntax.
