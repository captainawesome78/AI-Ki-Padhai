Python me web se data lene (GET) ya bhejne (POST) ke liye sabse famous library hai requests. Isse install karne ke liye likhiye:

Example: Simple GET Request

import requests

response = requests.get("https://api.github.com")

print(response.status_code)   # 200 ka matlab OK
print(response.headers)
print(response.text[:200])    # sirf pehle 200 characters print karega

Yahaan humne GitHub ke API ko ek GET request bheji hai. Agar status code 200 milta hai to iska matlab request sahi se gayi aur data aa gaya.

2Ô∏è‚É£ JSON Data ke sath kaam karna

Aaj kal zyada APIs data ko JSON format me bhejti hain. Python me hum JSON ko seedha dictionary me convert kar sakte hain: 

import requests

url = "https://api.github.com/users/octocat"
response = requests.get(url)

data = response.json()
print(data["login"])
print(data["public_repos"])
print(data["followers"])

Yahaan response.json() response text ko Python dictionary me badal deta hai. Ab hum us data me se specific cheezen nikaal sakte hain jaise username, repos ya followers.

3Ô∏è‚É£ POST Request (Server ko data bhejna)

Jab hume server ko koi information bhejni hoti hai (form fill karna, record banana), to hum POST request use karte hain: 

import requests

url = "https://httpbin.org/post"
data = {"name": "AIkiPadhai", "language": "Python"}

response = requests.post(url, json=data)
print(response.json())

Yahaan humne ek dictionary ko JSON format me bheja hai. Server ne hume wahi data wapas bhej diya.

4Ô∏è‚É£ Error Handling aur Timeouts

Internet connection kabhi kabhi slow ya down hota hai. Isliye hamesha apni request me error handling aur timeout lagana chahiye: 

import requests

try:
    response = requests.get("https://example.com", timeout=5)
    response.raise_for_status()
except requests.exceptions.Timeout:
    print("Request time out ho gaya!")
except requests.exceptions.RequestException as e:
    print("Error:", e)

timeout=5 ka matlab hai request 5 seconds tak wait karegi. Agar us time me reply nahi aaya to program error dega, par hang nahi karega.
