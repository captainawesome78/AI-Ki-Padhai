Conditions kya hote hain?

Kabhi-kabhi hume apne program me decision lena padta hai. Jaise agar temperature zyada hai to AC chalao, warna fan hi theek hai. Python me aise decisions lene ke liye if, elif, aur else statements use hote hain.

if Statement

temperature = 35

if temperature > 30:
    print("Bahut garmi hai!")

Output:

Bahut garmi hai!

if - else Statement

age = 16

if age >= 18:
    print("Aap vote de sakte hain")
else:
    print("Aap abhi vote nahi de sakte")

Output:

Aap abhi vote nahi de sakte

if - elif - else Statement

marks = 72

if marks >= 90:
    print("Grade A")
elif marks >= 75:
    print("Grade B")
elif marks >= 50:
    print("Grade C")
else:
    print("Fail")

Output:

Grade B

Nested if (if ke andar if)

age = 20
has_id = True

if age >= 18:
    if has_id:
        print("Entry allowed")
    else:
        print("ID dikhaiye")
else:
    print("Entry not allowed")

Output:

Entry allowed
