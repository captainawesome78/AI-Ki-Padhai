1. Simple Function

def greet():
    print("Hello AIkiPadhai!")

greet()

Output:

Hello AIkiPadhai!

2. Function with Parameters

def greet_user(name):
    print("Hello", name)

greet_user("Amit")

Output:

Hello Amit

3. Function with Return Value

def add(a, b):
    return a + b

result = add(10, 20)
print(result)

Output:

30

4. Default Parameters

def greet(name="Friend"):
    print("Hello", name)

greet()          # Hello Friend
greet("Amit")    # Hello Amit

5. Multiple Return Values

def arithmetic(a, b):
    return a+b, a-b, a*b

sum_, diff, prod = arithmetic(10, 5)
print(sum_, diff, prod)

Output:

15 5 50

6. Function Calling Another Function

def square(x):
    return x*x

def sum_of_squares(a, b):
    return square(a) + square(b)

print(sum_of_squares(2, 3))

Output:

13
