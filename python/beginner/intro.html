Python kya hai?

Python ek high-level, general-purpose programming language hai jo easy aur beginner-friendly hai. Iska syntax simple hai aur readability pe focus karta hai, isiliye beginners ke liye perfect hai.

Python ka use hota hai:

  • Web Development (Django, Flask)
  • Data Science aur Machine Learning (NumPy, Pandas, TensorFlow, PyTorch)
  • Automation & Scripting
  • Game Development (Pygame)

Python Install karna

1. Python.org pe jao aur latest Python version download karo. (Windows, macOS, Linux ke liye alag installers available hain.)

2. Installation ke time ek important checkbox hota hai: ‚ÄúAdd Python to PATH‚Äù ‚Äî ise tick karna zaroori hai.

3. Install hone ke baad, terminal ya command prompt open karo aur type karo: 

python --version

Agar properly install hua hai, to aapko kuch aisa output milega: 

Python 3.12.2

Python Run karna

Python run karne ke 2 popular tareeke hain:

  1. Interactive Mode (terminal mein directly run karna)
  2. Script Mode (ek file banakar run karna)

Example: Interactive Mode

>> print("Hello, World!")
Hello, World!

Visual Studio Code (VS Code) Install karna

Python likhne aur run karne ke liye ek IDE (Integrated Development Environment) ka use karna easy hota hai. Sabse popular IDE hai Visual Studio Code.

1. Download VS Code kijiye aur apne system par install kijiye.

2. Install karne ke baad, VS Code open kijiye aur Python extension install kijiye:

  • Left sidebar me ‚ÄúExtensions‚Äù (square icon) pe click kijiye
  • Search bar me Python likhiye
  • Microsoft ka official Python extension install kijiye

Apna Pehla Python Program (Hello World)

Ab ek folder open kijiye VS Code me aur ek naya file banao hello.py naam se.

File me likho:

print("Hello World from AIkiPadhai!")

Ab ise run karne ke liye:

  1. File save kijiye (Ctrl + S).
  2. VS Code ka terminal open kijiye (shortcut: Ctrl + `).
  3. Type kijiye:
python hello.py

Output:

Hello World from AIkiPadhai!
