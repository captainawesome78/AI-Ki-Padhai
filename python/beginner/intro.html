Python kya hai?

Python ek high-level, general-purpose programming language hai jo easy aur beginner-friendly hai. Iska syntax simple hai aur readability pe focus karta hai, isiliye beginners ke liye perfect hai.

Python ka use hota hai:

  • Web Development (Django, Flask)
  • Data Science aur Machine Learning (NumPy, Pandas, TensorFlow, PyTorch)
  • Automation & Scripting
  • Game Development (Pygame)

Python Install karna

1. Python.org pe jao aur latest Python version download karo. (Windows, macOS, Linux ke liye alag installers available hain.)

3. Install hone ke baad, terminal ya command prompt open karo aur type karo:

Agar properly install hua hai, to aapko kuch aisa output milega: 

Python 3.12.2

Python Run karna

Python run karne ke 2 popular tareeke hain:

  1. Interactive Mode (terminal mein directly run karna)
  2. Script Mode (ek file banakar run karna)

Example: Interactive Mode

>> print("Hello, World!")
Hello, World!

Example: Script Mode

Ek file banao hello.py naam se aur likho:

print("Hello from AIkiPadhai!")

Phir terminal mein run karo:

python hello.py

Output:

Hello from AIkiPadhai!
