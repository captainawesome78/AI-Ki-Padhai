Python List Methods

Python me lists ke useful functions aur operations

1. append()

fruits = ["apple", "banana"]
fruits.append("mango")
print(fruits)

Output:

['apple', 'banana', 'mango']

2. insert()

fruits = ["apple", "banana"]
fruits.insert(1, "orange")
print(fruits)

Output:

['apple', 'orange', 'banana']

3. remove()

fruits = ["apple", "banana", "mango"]
fruits.remove("banana")
print(fruits)

Output:

['apple', 'mango']

4. pop()

fruits = ["apple", "banana", "mango"]
fruits.pop()       # removes last element
print(fruits)
fruits.pop(0)      # removes first element
print(fruits)

Output:

['apple', 'banana']
['banana']

5. count()

numbers = [1,2,2,3,2]
print(numbers.count(2))  # 3

6. index()

numbers = [10,20,30,40]
print(numbers.index(30))  # 2

7. sort() & reverse()

numbers = [4,2,1,3]
numbers.sort()
print(numbers)     # [1,2,3,4]
numbers.reverse()
print(numbers)     # [4,3,2,1]
