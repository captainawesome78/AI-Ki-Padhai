String Methods in Python

Strings ko modify aur analyze karne ke common functions

Home Python AI About

.strip()

String ke aage/peeche ke extra spaces hata deta hai.

text = "   Hello AI!   "
print(text.strip())

Output:

Hello AI!

.split()

String ko ek list me todta hai based on separator (default = space).

text = "Python AI ML"
words = text.split()
print(words)

Output:

['Python', 'AI', 'ML']

.lower() aur .upper()

name = "AIkiPadhai"

print(name.lower())   # aikipadhai
print(name.upper())   # AIKIPADHAI

len()

String ki length (characters count) return karta hai.

text = "Python"
print(len(text))

Output:

6

Bonus: .replace()

String ke ek part ko dusre se replace karta hai.

text = "I love Java"
print(text.replace("Java", "Python"))

Output:

I love Python
