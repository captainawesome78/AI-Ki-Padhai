Variables kya hote hain?

Variable ek naam hota hai jisme hum data store karte hain. Isse hum program ke alag-alag parts me use kar sakte hain.

Example:

x = 10
name = "Ravi"
pi = 3.14

print(x)
print(name)
print(pi)

Output:

10
Ravi
3.14

Python me Data Types

Python me common data types ye hote hain:

  • int → Integer numbers (e.g. 5, 100, -42)
  • float → Decimal numbers (e.g. 3.14, -2.5)
  • str → String / Text (e.g. "Hello")
  • bool → True ya False values

Examples:

age = 25          # int
price = 19.99     # float
greeting = "Hi!"  # str
is_student = True # bool

print(type(age))
print(type(price))
print(type(greeting))
print(type(is_student))

Output:

<class 'int'>
<class 'float'>
<class 'str'>
<class 'bool'>

Variable Naming Rules

Python me variable banate waqt kuch rules follow karne hote hain:

  • Variable name me spaces nahi hote (use _ instead)
  • Sirf letters, numbers aur underscore allowed hai
  • Name number se start nahi hona chahiye
  • Case-sensitive hote hain (e.g. Age aur age alag hai)

Valid Examples:

my_name = "Amit"
age2 = 30
is_active = True

Invalid Examples (error aayega):

2age = 25
my name = "Rahul"
class = "Python"
