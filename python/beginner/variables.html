Variables kya hote hain?

Variables ek tarah ke containers hote hain jo data ko store karte hain. Python me variable banane ke liye bas value assign karte hain, type declare karne ki zarurat nahi hoti.

Example:

x = 10
y = "AIkiPadhai"
print(x, y)

Output:

10 AIkiPadhai

Data Types

Python me basic data types hote hain:

  • int → Integer numbers (jaise 5, 42, -10)
  • float → Decimal numbers (jaise 3.14, -0.5)
  • str → Strings (jaise "Python", "AI")
  • bool → Boolean (True ya False)

Example:

num = 42        # int
pi = 3.14       # float
name = "Python" # str
is_easy = True  # bool

print(num, pi, name, is_easy)

Output:

42 3.14 Python True
